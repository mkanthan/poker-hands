= \Project Euler Problem 54: Poker Hands

== Description

This is a simple ruby script to evaluate how many times Player 1 is the winner inside poker.txt. This is the solution to {Problem 54 on ProjectEuler.net}[https://projecteuler.net/problem=54].

== Project Setup

1) Run your standard bundle commands

  $ gem install bundler
  $ bundle

2) Run the full test suite - at the time of this writing, all tests should pass 100%

  $rake

3) Run the parser and display the answer.

  $rake poker
