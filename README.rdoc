= \Project Euler Problem 54: Poker Hands

== Description

This is a simple ruby script to evaluate how many times Player 1 is the winner inside poker.txt.

== Project Setup

1) Run your standard bundle commands

  $ gem install bundler
  $ bundle

2) Run +rake+ to run the test suite - at the time of this writing, all tests should pass 100%

3) Run +rake poker+ to run the parser and display the answer.
